# Fast Forward Fragments (FFF)

The Fast Forward Fragments (FFF) pipeline is a pipeline used by XChem as part of the Design, Make, Test and Analyse (DMTA) cycles to design new compounds based on a fragment hits. This documentation guides the user through the entire workflow.
